# ABYSS DOMINION v0.1.0-alpha.2

## テーマ：スキルシステム

### 追加
- 各モンスターに最大MP・現在MPを追加
- MPはダンジョン探索中に持ち越し
- 戦闘中のスキル選択メニュー
- レベル・ランクによるスキル習得判定
- 種族固有スキル
- 単体攻撃、多段攻撃、自己回復、全体回復、吸収攻撃
- MP不足時の使用制限
- ベッドでHP・MPを全回復

### セーブ互換
- schemaVersion 4
- 旧セーブには現在MPを自動追加
- 既存のモンスター・装備・探索記録を維持

### MP仕様
- 戦闘終了・階層移動では回復しない
- ベッド利用または全滅帰還時に全回復
