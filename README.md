Safariで古いJavaScriptが残り、v0.1.0-alpha.4.2の修正が反映されない問題へ対応しました。

- `main.js` のURLにバージョン番号を付与

- Safari/GitHub Pagesの古いキャッシュを強制的に回避
- 探索入力状態の初期化をさらに防御的に修正
- 画面上のバージョンを `v0.1.0-alpha.4.3` に更新

## 確認方法
適用後、ホーム画面に `REMAKE v0.1.0-alpha.4.3` と表示されていれば新しいコードが読み込まれています。
