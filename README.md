# ABYSS DOMINION

Current Version: **v0.2.1-alpha.2**

## Hotfix / Encounter Polish

### 修正
- 戦闘終了後にプレイヤーが消え、移動不能になる問題を修正
- 探索復帰時に `world.encountering` を必ず解除
- プレイヤーの移動経路と移動補間値を安全に初期化
- プレイヤー座標が不正な場合は開始地点へ復旧
- 探索ループを正常に再開

### 演出調整
- ランダムエンカウント演出を約1.7秒へ延長
- 揺れを約0.43秒
- 吸い込みを約0.92秒
- 暗転・閃光を約0.35秒

## 確認
戦闘終了後に「探索へ」を押し、
- 👑が表示される
- タップで移動できる
- 宝箱や深穴へ進める
ことを確認してください。
