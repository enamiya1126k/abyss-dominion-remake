# ABYSS DOMINION

Current Version: **v0.2.2-alpha.1**

## Save Migration & Equipment Management

### セーブ互換
- セーブスキーマを v6 へ更新
- 旧セーブを起動時に自動移行
- モンスター、装備、設定、探索記録を維持
- 旧装備データへ不足項目を自動追加

### 装備管理
- 表記を「武器・防具・アクセ」に統一
- キャラクター選択後、カテゴリタブで切り替え
- カテゴリごとに以下で並び替え
  - レア度
  - 総合能力
  - ATK
  - DEF
  - HP
  - SPD
  - 新しい順
  - お気に入り
  - 名前

### 容量
- 通常所持：500
- 予備BOX：30
- 王装保管庫：ボス限定装備専用・容量制限なし

### 容量超過時
1. 通常所持が満杯なら予備BOXへ自動転送
2. 予備BOXも満杯なら通常ドロップを自動換金
3. ボス限定装備は王装保管庫へ送り、絶対に失わない

予備BOXと王装保管庫は帰還中のみ整理できます。
