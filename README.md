# ABYSS DOMINION

Current Version: **v0.2.3-alpha.1**

## EXP Distribution Update

### 新仕様
- 戦闘不能になったモンスターは、その戦闘のEXPを獲得しない
- 本来パーティー全体へ配られる総EXPを、生存メンバーだけで均等分配
- 割り切れない余りEXPは、生存メンバーへ1ずつ配分
- 戦闘結果で「総EXP」「生存数」「戦闘不能：EXP 0」を表示

### 例
通常なら4体へ53EXPずつ配る戦闘の場合、総EXPは212。

- 4体生存：53 / 53 / 53 / 53
- 3体生存：71 / 71 / 70 / 戦闘不能0
- 2体生存：106 / 106 / 戦闘不能0 / 戦闘不能0
- 1体生存：212 / 戦闘不能0 / 戦闘不能0 / 戦闘不能0

旧セーブはそのまま使用できます。
