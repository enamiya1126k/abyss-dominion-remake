# ABYSS DOMINION

Current Version: **v0.2.0-alpha.1**

## 探索リビルド
- 初期階層を11×11へ縮小
- 階層に応じてマップが拡大
- 正方形・縦長・横長・洞窟・Y字マップ
- マップ上の追跡敵を廃止
- 8〜31歩のランダムエンカウント
- 敵接近の気配表示
- 宝箱を3〜8個へ増加
- 🪎・📦・🗃️・発光する✨📦
- 宝箱は階層ごとに一度きり
- 次階層を🕳️表示
- モンスターを仮絵文字化
- フィールドで自由に編成変更
- フィールドから装備管理
- ホームから到達済み階層を選択して再開

## 育成・ボス
- フィールドにEXP進捗を表示
- 戦闘後にEXPバーとレベルアップ演出
- 10階ごとに一度きりの「運命の4択」
- 限定LR武器・ボス加入・魔晶石・遠征用品から中身を見て選択
- 一度選ぶと他の報酬は取得不可
