# ABYSS DOMINION

Current Version: **v0.2.1-alpha.1**

## 探索フィール改善

- 全迷路の通路を原則2マス幅へ拡張
- 宝箱・ショップを開始地点と深穴の周囲へ置かない
- 宝箱へ向かう経路が深穴を横切らないよう経路探索を修正
- 深穴は明示的に選択したときだけ踏める
- ランダムエンカウント前の「気配」表示を廃止
- エンカウント時に画面揺れ、渦への吸い込み、暗転演出を追加
- プレイヤーの移動速度と補間を調整
- ショップ接触後に消えたミニマップへ描画して落ちる問題を修正
- DOM切り替え直後の描画ループを安全化
- ミニマップが存在しない場合の防御処理を追加

## テスト項目

1. 1階から探索を開始できる
2. 宝箱まで深穴を踏まずに移動できる
3. 🚪に触れてショップへ移動できる
4. ランダムエンカウント前に文章が表示されない
5. 画面揺れと吸い込み演出の後に戦闘が開始される
6. 戦闘終了後に探索へ戻れる
