# ABYSS DOMINION

Current Version: **v0.1.0-alpha.4.1**

## 緊急修正テーマ
スマホでテストプレイできるUIと、正しいラウンド制バトルへの修正。

## 修正
- GitHub PagesでCSSが読み込まれない大文字・小文字問題を修正
- 戦闘画面をスマホ1画面向けに再調整
- 行動順パネルを追加
- 敵味方全員をSPD順に並べる
- 敵味方とも1ラウンドにつき1回だけ行動
- 敵が最速の場合は敵から行動
- 同SPD時はラウンドごとに順番を抽選
- 戦闘不能キャラクターは残り行動をスキップ
- 敵の攻撃対象を生存メンバーから選択
- ガードをラウンド内で有効化
- 毒・炎上・クールダウンをラウンド終了時に処理
- AUTOも同じ行動順ルールで進行
- 逃走失敗で現在キャラクターの行動を消費

## 重要
この更新では新しい収集・報酬機能を増やさず、テストプレイの成立を優先しています。

## 次
実機プレイ後の感想を基に、UI・戦闘テンポ・数値バランスを修正します。
