## GOLD設計

手動帰還1階分を経済の基準値とし、深層でも各コンテンツの価値が消えない比率へ統一。

- 通常戦：3%
- 初回階層ボス：15%
- 宝箱：10%
- エリート追加報酬：14%
- 第二世界の高リスクGOLDイベント：10〜15%
- 深淵の鍵購入：4%
- チームバトル勝利：試練段階に応じ8.8〜20%
- 十神・Deep Abyss勝利：顕現率に応じ22〜40%

敗北によるGOLD稼ぎを防ぐため、特殊戦のGOLDは勝利時だけ獲得する。
