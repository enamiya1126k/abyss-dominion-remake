# CHANGELOG

## v0.9.15-alpha.3 Part 4 — 第二世界エリートモンスター

### Added
- 1001階以降に低確率で出現する深淵エリート個体
- 5種類の変異能力：巨大化・狂暴・高速・再生・魔力暴走
- 歪んだ裂け目から確定で発生するエリート戦闘
- 専用WARNING演出、紫色の戦闘UI、ELITEバッジ
- エリート専用のゴールド・魔晶石・深淵の鍵報酬
- 遭遇数、討伐数、能力別・種族別討伐記録

### Compatibility
- セーブスキーマを `24` へ更新
- 既存セーブへエリート記録を自動追加


## v0.9.15-alpha.3 Part 3 — 第二世界ランダムイベント

### Added
- 1002階以降の一部階層で発生する第二世界ランダムイベント
- 「深淵の祭壇」「顔のない商人」「深淵結晶の群生」「歪んだ裂け目」
- 各イベントの3択と、回復・資源・エリート戦闘への分岐
- 同じ階層でイベントが重複再生されない解決済み記録

### Compatibility
- セーブスキーマを `23` へ更新
- 既存セーブに第二世界イベント履歴を自動追加
